# ☁️ IBIMS Cloud Infrastructure

The backend for IBIMS uses **Google Apps Script (GAS)** to provide a serverless REST API that interfaces with Google Sheets as a real-time data lake.

## 🛠️ Deployment Steps
1. Create a new Google Sheet.
2. Navigate to **Extensions > Apps Script**.
3. Paste the contents of `GoogleAppsScript.js` into the script editor.
4. Click **Deploy > New Deployment**.
5. Select **Web App** as the deployment type.
6. **Crucial:** Set "Who has access" to **Anyone**. 
7. Authorize the permissions and copy the **Deployment ID**.

## 📊 Data Structure
The script automatically appends data to the next available row with the following headers:
- **Timestamp:** Generated by the Google Server ($MM/DD/YYYY HH:MM:SS$).
- **Event Status:** "Placed" or "Picked".
- **Raw Sensor Value:** The ADC spike value captured during the event.

## 🔗 Endpoint Format
The ESP8266 sends a standard HTTPS GET request:
`https://script.google.com/macros/s/[ID]/exec?status=[STATUS]&val=[VALUE]`
